# الخلاصة الحسناء — تطبيق متابعة أذكار الصباح والمساء

تطبيق ويب بسيط (HTML/CSS/JS بدون أي إعداد بناء Build) لمتابعة أذكار الصباح والمساء، بنفس ترتيب واختيار كتاب **"الخلاصة الحسناء في أذكار الصباح والمساء"** للشيخ صالح بن عبدالله العصيمي.

## المزايا
- نص كل ذكر، مع **العدد المطلوب** و**العدد المُنجز** وعلامة ✓ عند إتمامه.
- عدّاد تفاعلي لكل ذكر (+/-) وزر "تعليم الكل كمُنجز".
- **تسجيل يومي** تلقائي محفوظ في متصفحك (localStorage) بدون أي خادم أو حساب.
- **نسخة احتياطية**: تصدير كل سجلاتك كملف JSON، واستيرادها لاحقًا أو على جهاز آخر.
- **إحصائيات المتابعة والالتزام**: عدد الأيام المتتالية، أفضل التزام، نسبة الإنجاز آخر 30 يومًا، خريطة حرارية، ورسم بياني أسبوعي.
- تصميم يتغيّر تلقائيًا بين طابع الصباح (ذهبي) والمساء (نيلي).
- **قابل للتثبيت كتطبيق (PWA)** على الجوال أو الحاسوب، ويعمل بدون اتصال بالإنترنت بعد أول زيارة.

## الملفات
```
index.html     ← الصفحة الرئيسية
styles.css     ← التصميم
app.js         ← منطق التطبيق (العدّادات، التسجيل، الإحصائيات، النسخ الاحتياطي)
data.js        ← نصوص الأذكار وعدد التكرار المطلوب لكل ذكر
manifest.json  ← بيان التطبيق (PWA): الاسم، الأيقونات، الألوان
sw.js          ← عامل الخدمة (Service Worker) لتفعيل العمل بدون اتصال
icons/         ← أيقونات التطبيق بأحجام مختلفة
```

لا حاجة لأي تثبيت أو npm أو خطوة بناء — الملفات الثلاثة تعمل مباشرة في أي متصفح أو استضافة استاتيكية.

## النشر على GitHub

1. أنشئ مستودعًا (Repository) جديدًا على حسابك في GitHub، مثلاً باسم `adhkar-app`.
2. ارفع الملفات الأربعة (`index.html`, `styles.css`, `app.js`, `data.js`) إلى المستودع:
   ```bash
   git init
   git add .
   git commit -m "أذكار الصباح والمساء - نسخة أولى"
   git branch -M main
   git remote add origin https://github.com/<اسم-حسابك>/adhkar-app.git
   git push -u origin main
   ```
3. (اختياري) لتفعيل **GitHub Pages** مباشرة من المستودع:
   - اذهب إلى **Settings → Pages**.
   - في **Source** اختر الفرع `main` والمجلد `/ (root)`.
   - احفظ، وسيصبح الموقع متاحًا خلال دقائق على رابط مثل:
     `https://<اسم-حسابك>.github.io/adhkar-app/`

## النشر على Netlify

**الطريقة الأسهل (سحب وإفلات):**
1. ادخل إلى [app.netlify.com](https://app.netlify.com) وسجّل الدخول.
2. من لوحة التحكم اختر **"Add new site" → "Deploy manually"**.
3. اسحب مجلد المشروع بأكمله (الذي يحوي الملفات الأربعة) وأفلته في الصفحة.
4. سيُنشر الموقع فورًا برابط عشوائي، ويمكنك تغييره من **Site settings → Change site name**.

**الطريقة عبر GitHub (نشر تلقائي عند كل تحديث):**
1. بعد رفع المشروع على GitHub كما في الخطوات أعلاه.
2. من Netlify اختر **"Add new site" → "Import an existing project"** واربط حساب GitHub.
3. اختر مستودع `adhkar-app`.
4. اترك خانة **Build command** فارغة، واجعل **Publish directory** يساوي `.` (المجلد الجذر).
5. اضغط **Deploy site**.

## تثبيت التطبيق على الجوال (PWA)
بعد نشر الموقع على GitHub Pages أو Netlify (كلاهما يوفّر HTTPS تلقائيًا، وهو مطلوب لعمل عامل الخدمة):
- **على أندرويد (Chrome)**: افتح الرابط ثم اضغط قائمة ⋮ → "إضافة إلى الشاشة الرئيسية".
- **على آيفون (Safari)**: افتح الرابط ثم اضغط زر المشاركة 📤 → "إضافة إلى الشاشة الرئيسية".
- **على الحاسوب (Chrome/Edge)**: ستظهر أيقونة تثبيت ⊕ في شريط العنوان.

بعد التثبيت سيعمل التطبيق بواجهة مستقلة (بدون شريط المتصفح) وبدون اتصال بالإنترنت.

## ملاحظات
- كل البيانات تُحفظ محليًا في متصفح المستخدم (لا يوجد خادم خلفي)، لذا يُنصح كل مستخدم بأخذ نسخة احتياطية دوريًا من تبويب "النسخ الاحتياطي" حفاظًا على سجل متابعته.
- نصوص الأذكار مأخوذة من السنة الثابتة عن النبي ﷺ، مرتّبة وفق اختيار كتاب الخلاصة الحسناء؛ يُنصح بمراجعة الكتاب الأصلي أو الشرح الصوتي للشيخ العصيمي للاستزادة.
